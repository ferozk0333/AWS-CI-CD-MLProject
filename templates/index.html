Student Exam Performance Indicator

Test Application

Learning Outcome

Understanding modular programming approach to build data solutions for clients.

Problem Statement

The CAT is an MBA entrance exam for admission to B-schools in India. An eminent test prep institute has data of 1000 students with features including gender, race, parental education, lunch type, test prep course, writing score, and reading score. The institute wants to understand how these factors influence a student's performance in the math section of the test.

Proposed Solution

As a data scientist, the aim is to first determine if there is a significant relationship between the independent variables and the target feature. If so, the following approach will be taken:

  1. Understanding the Problem Statement: Analyze the provided data to identify patterns and correlations between student attributes and math scores.
  2. Data Collection from data source
  3. Exploratory Data Analysis: Handling missing values, duplicates, balancing dataset, basic statistics and more.
  4. Data Pre-Processing: Cleanse and prepare the data for machine learning modeling, standard scaling.
  5. Model Training: As this is a regression problem, test algorithms such as linear regression, decision tree regression, to build predictive models.
  6. Hyperparameter Tuning: Optimize model performance by tuning hyperparameters and identify best parameters.
  7. Build Application and Deploy It: Develop an end-to-end application that allows users to input student data and receive predicted math scores.

Technical Details

  • Deployed on AWS Elastic Beanstalk with CI CodePipeline
  • Front end: Flask
  • Major modules: Numpy, Pandas, Sklearn, Matplotlib.pyplot
  • Regression problem - Regression algorithms used with hyperparameter tuning
